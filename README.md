# 专业图片压缩工具 (CompressLab)

一个基于 Next.js 14、TypeScript 和先进压缩算法构建的现代化专业级图片压缩和格式转换工具。具有智能压缩策略、精确大小控制、无缝格式转换功能和完善的SEO优化。

🌐 **在线体验**: [https://tinyimgtool.com](https://tinyimgtool.com)  
📚 **项目仓库**: [https://github.com/boshiii123/ceode4](https://github.com/boshiii123/ceode4)

## 🚀 核心功能

### 智能压缩引擎
- **智能压缩算法**：基于文件大小的多层压缩策略
  - 大文件 (>5MB)：激进压缩，智能尺寸缩放
  - 中等文件 (2-5MB)：平衡压缩，保持质量
  - 小文件 (<2MB)：轻度压缩，保留细节
- **精确大小控制**：目标特定文件大小（100KB、200KB、300KB），高精度压缩
- **二分搜索优化**：使用二分搜索算法进行高级质量优化
- **智能参数预测**：基于压缩需求的智能初始质量计算
- **渐进式压缩**：分层方法，具有最佳结果的回退策略

### 格式转换系统
- **多格式支持**：WebP、JPEG、PNG 和 AVIF 格式之间的转换
- **转换模式**：
  - 有损转换：更小的文件大小
  - 无损转换：最大质量保留
- **单个和批量转换**：转换单个图片或整批图片
- **实时格式切换**：每个图片的快速格式转换按钮
- **智能转换逻辑**：在优化目标格式的同时保持质量

### 专业用户界面
- **现代玻璃态设计**：美丽的渐变背景，带有背景模糊效果
- **响应式布局**：针对桌面、平板和移动设备优化
- **拖拽上传**：直观的文件上传，带有视觉反馈
- **实时进度跟踪**：详细的进度指示器，逐文件状态显示
- **专业预览系统**：紧凑缩略图，包含全面的文件信息
- **交互式控件**：带有图标和悬停效果的专业按钮

### 批量处理功能
- **多文件上传**：同时处理多个图片（最多20个）
- **批量压缩**：将压缩设置应用于所有图片
- **批量格式转换**：将所有图片转换为相同格式
- **独立控制**：独立微调每个图片
- **批量下载**：一次下载所有处理过的图片
- **重置功能**：将图片恢复到原始压缩状态

### 🔒 隐私保护
- **100% 客户端处理**：所有图片压缩在浏览器中完成
- **数据不上传**：您的图片永远不会离开您的设备
- **无服务器存储**：我们不收集、存储或分析您的图片

## 🎯 详细功能文档

### 压缩算法详情

#### 智能初始压缩
工具根据文件大小自动选择压缩参数：

- **大文件 (>5MB)**：
  - 初始质量：0.6
  - 最大尺寸：1600px
  - 目标：1MB 中间大小
  
- **中等文件 (2-5MB)**：
  - 初始质量：0.7
  - 最大尺寸：用户定义
  - 目标：0.8MB 中间大小（保守策略）
  
- **小文件 (<2MB)**：
  - 初始质量：0.8（固定值）
  - 最大尺寸：用户定义
  - 保留原始特征

#### 精确目标压缩
对于特定大小目标（100KB、200KB、300KB）：

1. **智能质量预测**：基于所需压缩比计算初始质量
2. **二分搜索优化**：使用12次尝试的二分搜索获得最佳质量
3. **尺寸缩减**：当质量优化不足时的智能尺寸缩放
4. **回退策略**：使用不同参数的多次压缩尝试
5. **严格目标控制**：确保不超过目标大小

### 格式转换系统

#### 支持的格式
- **WebP**：现代格式，出色的压缩效果，25-35% 比 JPEG 更小
- **AVIF**：下一代格式，卓越的压缩效果，比 JPEG 小 50%
- **JPEG**：通用兼容性，良好的压缩效果，适合照片
- **PNG**：无损格式，适用于图形和透明度
- **GIF、BMP、TIFF**：支持输入，可转换为现代格式

#### 转换模式
1. **有损模式**：从压缩版本转换，文件更小
2. **无损模式**：从原始文件转换，最大质量

### 用户界面组件

#### 上传区域
- **视觉状态**：空闲、悬停和处理状态的不同外观
- **进度集成**：实时进度条，逐文件跟踪
- **文件验证**：自动格式检测和验证
- **错误处理**：优雅的错误处理，用户反馈

#### 设置面板
- **选项卡界面**：压缩和转换工具的独立选项卡
- **交互控件**：实时值显示的范围滑块
- **快速操作**：常用大小的一键压缩按钮
- **批量操作**：同时处理所有图片的工具

#### 预览系统
- **紧凑缩略图**：192px 宽度的专业缩略图
- **全面信息**：文件大小、格式、压缩比
- **操作按钮**：下载、转换、重置和删除选项
- **状态指示器**：处理状态的视觉反馈
